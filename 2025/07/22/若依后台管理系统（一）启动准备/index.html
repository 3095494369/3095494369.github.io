启动项目

版本要求：

JDK >= 1.8 (推荐1.8版本)
Mysql >= 5.7.0 (推荐5.7版本)
Redis >= 3.0
Maven >= 3.0
Node >= 12

前端：

# 进入项目目录
cd ruoyi-ui

# 安装依赖
npm install

# 强烈建议不要用直接使用 cnpm 安装，会有各种诡异的 bug，可以通过重新指定 registry 来解决 npm 安装速度慢的问题。
npm install --registry=https://registry.npmmirror.com

# 本地开发 启动项目
npm run dev

后端修改必要的配置和开启启动项：

  1. 修改数据库连接，编辑resources目录下的application-druid.yml
# 数据源配置
spring:
    datasource:
        type: com.alibaba.druid.pool.DruidDataSource
        driverClassName: com.mysql.cj.jdbc.Driver
        druid:
            # 主库数据源
            master:
                url: 数据库地址
                username: 数据库账号
                password: 数据库密码
  1. 修改服务器配置，编辑resources目录下的application.yml
# 开发环境配置
server:
  # 服务器的HTTP端口，默认为80
  port: 端口
  servlet:
    # 应用的访问路径
    context-path: /应用路径
  1. 开启redis服务（本地或虚拟机都可以）
  2. 修改redis配置
redis:
  # 地址--我的是虚拟机，本地是localhost
  host: 192.168.239.128
  # 端口，默认为6379
  port: 6379
  # 数据库索引
  database: 0
  # 连接超时时间
  timeout: 10s
